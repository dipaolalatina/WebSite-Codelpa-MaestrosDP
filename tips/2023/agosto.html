“Protección para Rejas y Metales.”

1 Protección de acero.

Para lograr un buen resultado en la protección del acero se debe aplicar un fondo con Anticorrosivo Maestranza Sipa y para la terminación el Esmalte sintético Sipa.

2 Eliminar resto de pintura, óxido, aceites o grasas.

Antes de proteger y embellecer el acero se deben eliminar las pinturas sueltas, óxidos, aceites o grasas para un buen resultado.

3 Adherencia de esquema anticorrosivo.

Para lograr una buena adherencia del Esquema Anticorrosivo, primero hay que eliminar aceites y grasas, lavando con agua y detergentes líquidos biodegradables y posteriormente secando la superficie

4 Lijar para mayor adherencia.

Se deben eliminar las pinturas sueltas y óxidos mediante lijado y/o herramientas mecánicas para permitir una mayor adherencia y mejor resultado.

5 Antes de aplicar...

Antes de aplicar el Esquema Anticorrosivo al acero, se deben tratar las soldaduras para una buena adherencia.

6 Como cubrir manchas de humedad.

Para cubrir manchas en interiores provocadas por humedad, humos y/o luces se debe aplicar una mano de Óleo Opaco Sipa en la superficie, dejar secar y luego aplicar Látex o Esmalte al Agua Sipa.

7 Consultar el clima.

Para un buen resultado en la aplicación del Esquema Anticorrosivo en acero, primero se debe consultar las condiciones ambientales.

8 Lavar con detergente biodegradable.

Para ZINC ALUM y Galvanizado solo se debe lavar con detergente biodegradable líquido, enjuagar, secar y aplicar dos manos de ACRIZINC Sipa.
